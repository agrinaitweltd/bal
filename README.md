# Goodfoot Ballet Dance & Music Academy

A premium, modern React + TypeScript website for **Goodfoot Ballet Dance & Music Academy** located on Mawanda Rd, Kampala, Uganda.

## Tech Stack

- **React 18** + **TypeScript** (TSX)
- **Vite** – fast development build tool
- **Framer Motion** – smooth animations
- **React Router v6** – client-side routing
- **CSS Modules** – scoped, structured styles

## Pages

| Page | Route | Description |
|------|-------|-------------|
| Home | `/` | Hero, stats, about preview, classes, why-us, testimonials, CTA, gallery preview |
| About | `/about` | Academy story, achievements, mission/vision, instructors, philosophy |
| Classes | `/classes` | All 8 class categories with levels, schedules, and pricing |
| Services | `/services` | 6 service offerings + how-it-works process |
| Gallery | `/gallery` | Filterable animated gallery with lightbox |
| Contact | `/contact` | Contact form, WhatsApp CTA, map placeholder, social links |

## Getting Started

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
npm run preview
```

## Deployment

Configured for **Vercel** deployment via `vercel.json`.

Push to GitHub and connect to Vercel for instant deployment.
